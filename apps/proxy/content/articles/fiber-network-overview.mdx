Fiber Network brings Lightning Network-style payment channels to the Nervos CKB ecosystem. This technical overview explains how Fiber enables fast, cheap payments while inheriting CKB's security.

## What is Fiber?

Fiber is a layer-2 payment protocol built on CKB that enables:
- Instant payments without waiting for block confirmations
- Micropayments as small as a fraction of a cent
- Multi-hop routing across the network
- Cross-chain interoperability with Bitcoin Lightning

Unlike traditional payment channels, Fiber leverages CKB's unique cell model and programmable state for enhanced flexibility.

## Technical Architecture

Fiber implements the BOLT (Basis of Lightning Technology) specifications adapted for CKB's architecture:

### Payment Channels
- **Funding Transaction**: Locks CKB in a 2-of-2 multisig cell
- **Commitment Transactions**: State updates signed by both parties
- **HTLCs**: Hash Time-Locked Contracts enable multi-hop payments
- **Force Close**: On-chain settlement when parties disagree

### Routing
Fiber uses source routing with encrypted payment paths. Senders specify the complete route, and intermediate nodes cannot determine the payment's origin or destination.

### UDT Support
Fiber uniquely supports User Defined Tokens (UDTs), enabling:
- Lightning-fast stablecoin transfers
- NFT micropayments
- Custom token economies

## CKB Advantages

### Programmability
CKB's Turing-complete scripts allow complex channel logic:
- Custom dispute resolution
- Advanced HTLC variations
- Integration with DeFi protocols

### Economic Model
CKB's state rent model ensures long-term channel sustainability. Unlike other networks, state growth has predictable costs.

### Interoperability
Fiber's design supports:
- BTC Lightning swaps via HTLCs
- CKB-Bitcoin atomic swaps
- Cross-hub liquidity sharing

## Running a Node

Setting up a Fiber node requires:
1. CKB node synchronization
2. Fiber node installation
3. Channel liquidity allocation
4. Peer connections

The reference implementation is written in Rust for performance and safety.

## Use Cases

- **Micropayments**: Pay per article, API call, or second of content
- **Remittances**: Near-instant cross-border transfers
- **Gaming**: Real-time in-game economies
- **IoT**: Machine-to-machine payments

Fiber represents the next evolution of payment channels. By combining Bitcoin's proven Lightning technology with CKB's flexibility, it enables use cases impossible on other chains.

As the network grows, expect to see:
- More exchange integrations

The future of payments is fast, cheap, and interoperable—and Fiber is leading the way.
